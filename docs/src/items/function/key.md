## 描述

古老的钥匙: 有些年头的东西,或许可以找到配对的锁

城东乾府的钥匙/城东坤府的钥匙: 手持钥匙右击门边的箱子  即可打开府邸的大门

铜钥匙/银钥匙/金钥匙: 可以打开副本的奖励箱。能够让使用者取得神兵利器，开创未来新页的传说之钥。

## 获取方式

古老的钥匙: 钓鱼获得

城东乾府的钥匙/城东坤府的钥匙: 前往忠烈祠购买城东乾府/城东坤府后获得

铜钥匙/银钥匙/金钥匙: 与NPC进行交易获得/金钥匙可用400功勋值兑换
